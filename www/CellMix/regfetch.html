Advanced Fetch for Registry

Usage

regfetch(regobj, key, msg = NULL, error = TRUE, exact = FALSE, multiple = FALSE, 
  verbose = FALSE)

Arguments

regobj
registry object.
key
entry accession key.
msg
error prefix.
error
logical that indicates if an error should be thrown if the key is not found or if matches multiple entries (when multiple=FALSE).
exact
logical that indicates if the one looks for exact match(es).
multiple
logical that indicates if the key can return multiple matches, or NA if error=FALSE, or throw an error if error=TRUE.
verbose
logical that toggles verbosity.

Description

Retrieve entries from a registry object.
