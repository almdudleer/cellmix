Mapping Bioconductor Annotation packages to GEO GPL Identifiers

Usage

gpl2bioc(x)

bioc2gpl(x)

Arguments

x
character vector of GPL ids or Bioconductor packages to map.

Value

a character vector

Description

gpl2bioc and bioc2gpl maps a vector of GEO GPL ids into Bioconductor annotation packages, and vice versa.
