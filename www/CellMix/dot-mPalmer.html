Marker List for Immune Cells - Palmer et al. (2006)

Usage

.mPalmer(id = c("SYMBOL", "LUID"), score = c("Corr", "Fold"), threshold = 0)

Arguments

id
original identifier to use.
score
name of the column from the IRIS dataset to use as a score.
threhold
minimum value threshold for filtering on scores.

Description

CellMix access key: “Palmer”

Marker list created from the Palmer dataset (Palmer et al. (2006)). It contains markers for B-cells, CD4+ and CD8+ T-cells, lymphocytes and granulocytes.

Details

The MarkerList object is generated by the internal function .mPalmer, to which arguments can be passed via MarkerList('Palmer', ...) to build lists using different identifiers and/or filtering threshold.

However, it is recommended to rather use the function convertIDs to convert the primary identifiers into other identifiers, using up-to-date annotation packages.

References

Palmer C, Diehn M, Alizadeh Aa and Brown PO (2006). "Cell-type specific gene expression profiles of leukocytes in human peripheral blood." _BMC genomics_, *7*, pp. 115. ISSN 1471-2164, , .
