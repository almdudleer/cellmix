Partial Sanitizer for xtable

Usage

xsanitize(skip)

Arguments

skip
character string that contain the character not to sanitize.

Value

a function that can be used in argument sanitize.text.function of print.xtable.

Description

Text sanitizing function to use in argument sanitize.text.function of print.xtable. This function is used in CellMix to format the content internal registries for inclusion in vignettes, as they contain citation commands (\cite).

Examples

library(xtable)

print(xtable(gedAlgorithmInfo(show=FALSE), citet=TRUE), sanitize.text.function=xsanitize("\\{}$"))
