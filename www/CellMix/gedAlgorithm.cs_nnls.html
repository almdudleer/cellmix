Partial Cell-Specific Expression by Nonnegative Least-Squares

Usage

gedAlgorithm.cs_nnls(..., rescale = TRUE)

Arguments

...
extra arguments passed to .fcnnls.
rescale
logical used when estimating proportions from signatures, that indicates if the esti,ated coefficients should be scaled to sum up to one (TRUE) or left as estimated by the linear regression (FALSE). This scaling is performed after the coefficients have been forced to be nonnegative.

Description

Estimates cell-specific signatures given known cell proportions, using the fast combinatorial nonnegative least-square as the ged algorithm nnls. It is included in the CellMix package for test/experimental purposes.

Details

All regressions are fitted using the function .fcnnls.

Examples

# random target matrix
x <- rmatrix(100, 20)

# random cell proprtions
p <- rmatrix(3, 20)

# deconvolve using nonnegative least-squares
res <- ged(x, p, 'cs-nnls')
