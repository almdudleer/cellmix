Wrapping GED Algorithms

Usage

gedWrapper(key)

Arguments

key
Name of the GED algorithm to be wrapped. It should be the name of a registered algorithm as returned by gedAlgorithm, or a GED algorithm object (i.e. an instance of GEDStrategy-class).

Value

a function with attribute 'algorithm' set to the value of key

Description

This function creates a wrapper function for calling the function ged with a given GED algorithm.

Examples

# wrap 'qprog' algorithm from Gong et al. (2011)
qp <- nmfWrapper('qprog')

qp



function (..., data = NULL, exact = TRUE, log = NULL) 
{
    ca <- match.call()
    args <- formals()
    .call <- .checkArgs(ca, args)
    e <- parent.frame()
    eval(.call, envir = e)
}
