Processing of Phenotypic Annotation for Dataset GSE3649

Usage

.GSE3649_pdata(eset, file = "GSE3649_pdata.csv")

Arguments

eset
ExpressionSet object from GSE3649 [GPL3121], with Nepal samples filtered out.
file
Phenotypic annotation file (.csv) as downloaded from http://genome-www.stanford.edu/normalblood/.

Source

http://genome-www.stanford.edu/normalblood/

Value

a data.frame

Description

Function used to create the additional phenotypic annotation data GSE3649_fdata for the GSE3649 expression data.

See also

Other Whitney: .GSE3649_convert
