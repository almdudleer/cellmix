HaemAtlas Marker List for Immune Cells - Watkins et al. (2009)

Usage

.mHaemAtlas(id = c("ILMN", "nuID", "ENTREZID", "SYMBOL", "_ILMN", "_ENSEMBL", "_SYMBOL"))

Arguments

id
original identifier to use.

Description

CellMix access key: “HaemAtlas”

Marker list created from the HaemAtlas dataset (Watkins et al. (2009)). It contains markers for CD4+ and CD8+ T-cells, lymphocytes, monocytes, B-cells, NK cells, and granulocytes.

Details

The MarkerList object is generated by the internal function .mHaemAtlas, to which arguments can be passed via MarkerList('HaemAtlas', ...) to build lists using different identifiers.

However, it is recommended to rather use the function convertIDs to convert the primary identifiers into other identifiers, using up-to-date annotation packages.
