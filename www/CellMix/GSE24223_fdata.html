Description

Supplemental Table 5 from Grigoryev et al. (2010) containing average gene expression values for cytometry antigen markers, within each group of samples. Groups are defined by the pair of experimental factors (Type, Time) (i.e. Cell type x Time of blood draw).

.GSE24223_fdata is the function used to create the GSE24223_fdata data.
