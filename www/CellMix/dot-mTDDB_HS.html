Human Tissue Specific Genes from the TissueDistributionDB Database

Usage

.mTDDB_HS(percent = 99)

Arguments

percent
minimum percentage expression value required for a gene to be included as a marker.

Description

CellMix access key: “TissueDistributionDB_HS”

Marker list for Human tissues based on Unigene EST clusters, downloaded from the TissueDistributionDB database (Kogenaru et al. (2009)).

Details

The MarkerList object is generated by the internal function .mTDDB_HS, to which arguments can be passed via MarkerList('TDDB_HS', ...) to build lists using a different filtering threshold.

References

Kogenaru S, Val C, Hotz-Wagenblatt A and Glatting K (2009). "TissueDistributionDBs: a repository of organism-specific tissue-distribution profiles." _Theoretical Chemistry Accounts_, *125*(3-6), pp. 651-658. ISSN 1432-881X, , .

Examples

MarkerList('TDDB_HS')
