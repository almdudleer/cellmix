Building Constraint Matrices

Usage

maxConstraint(n, i_max = 1, alpha = 1)

Arguments

n
number of components.
i_max
index of the component that must be greater that all others.
alpha
minimum ratio between the k-th and all the other components.

Description

These are utility functions to work with constraint matrices, which define linear equality and inequality constraints in optimization problems.

maxConstraint creates a constraint matrix that specifies that a given component must be greater than all the others. It is used, in particular, in the deconvolution method from Gong et al. (2011) to enforce which is“qprog”

Details

maxConstraint creates a constraint matrix that specifies that a component k must be greater than all the others:

x_k >= alpha * x_i,

  
      
for all 1<= i <= n, where
  alpha is a fixed parameter is the minimum
  ratio between the k-th and all the others
  components, i.e. it controls the degree of differences
  between these components.

References

Gong T, Hartmann N, Kohane IS, Brinkmann V, Staedtler F, Letzkus M, Bongiovanni S and Szustakowski JD (2011). "Optimal deconvolution of transcriptional profiling data using quadratic programming with application to complex clinical blood samples." _PloS one_, *6*(11), pp. e27156. ISSN 1932-6203, , .
