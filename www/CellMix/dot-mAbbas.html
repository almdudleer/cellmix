Optimised Set of Marker Genes for Immune Cells

Usage

.mAbbas(sparsity = 0.8, id = c("AFFY", "ENTREZID", "SYMBOL"))

Arguments

sparsity
sparsity threshold (see sparseness)
id
original identifier to use.

Source

Supplementary Table 1 of Abbas et al. (2009) Found at: doi:10.1371/journal.pone.0006098.s001 (0.11 MB PDF)

Description

CellMix access key: “Abbas”

Curated list of marker probesets for immune cell types from the Abbas data (Abbas et al. (2009)). It contains Affymetrix probesets of markers for B cells, dendritic cells, monocytes, neutrophils, NK cells and T cells, as well as for the lymphoid and myeloid lineages, and the general category of all these immune cells.

Details

The MarkerList object is generated by the internal function .mAbbas, to which arguments can be passed via MarkerList('Abbas', ...) to build lists, using different original identifiers and filtering threshold.

However, it is recommended to rather use the function convertIDs to convert the primary identifiers into other identifiers, using up-to-date annotation packages.

The markers are ordered in each cell type by decreasing sparsity value, i.e. from the more to the less cell type-specific, computed from the optimised signature matrix Abbas.
