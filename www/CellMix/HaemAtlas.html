HaemAtlas Dataset - Immune Cells

Source

http://www.ncbi.nlm.nih.gov/pmc/articles/PMC2680378/bin/blood-2008-06-162958_TableS5.xls

Description

The dataset contains markers for CD4+ and CD8+ T-cells, lymphocytes, monocytes, B-cells, NK cells, and granulocytes obtained from the HaemAtlas Watkins et al. (2009).

Details

The dataset was created using the internal function .HaemAtlas.

The probe ids provided in the supplementary data file are somehow incorrect, and do not correspond to the Illumina ids found in the illuminaHumanv2.db annotation package. "Correct" probe ids were recovered from their nucelotide sequences, via their nuIDs.
