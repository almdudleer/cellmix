Complete Blood Count for Dataset GSE20300

Source

Extracted from Supplementary Table 3 of Shen-Orr et al. (2010)

Description

The data was extracted from the PDF file and reordered in the same order as the microarray dataset GSE20300, assuming that the GEO samples' order within each group (Stable or ACR) is consistent with the samples' order in the PDF file.

References
