TiGER - Human Tissue Specific Genes

Usage

.mTIGER(score = 5, log10P = 3.47853)

Arguments

score
minimum score for a gene to be included in the list
log10P
minimum value for -log10(P-value) for a gene to be included in the list

Description

CellMix access key: “TIGER”

Marker list created from the TIGER dataset (Liu et al. (2008)). The TiGER database is based on UniGene EST distribution profiles. The markers are defined by thresholds on enrichment scores and their associated log10 p-values.

Details

The MarkerList object is generated by the internal function .mTIGER, to which arguments can be passed via MarkerList('TIGER', ...) to build lists using different filtering threshold.

The function convertIDs can be used to convert the primary identifiers into other identifiers (e.g. ENTREZ ids or microarray probe ids), using up-to-date mappings from Bioconductor annotation packages.

References

Liu X, Yu X, Zack DJ, Zhu H and Qian J (2008). "TiGER: a database for tissue-specific gene expression and regulation." _BMC bioinformatics_, *9*, pp. 271. ISSN 1471-2105, , .

Examples

MarkerList('TIGER')
