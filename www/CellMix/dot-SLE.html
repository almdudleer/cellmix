Processing for SLE Data

Usage

.SLE(file = "TableS2.txt")

Arguments

file
input file. The file would be the XLS (actually tab separated) file provided as Supplementary Table S2 from Abbas et al. (2009).

Value

an ExpressionSet object

Description

Function used to create the SLE data.
