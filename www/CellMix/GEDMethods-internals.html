Register CellMix Deconvolution Methods

Usage

setGEDMethod(key, ...)

removeGEDMethod(key, ...)

Arguments

key
accession key.
...
extra registry fields describing the method, or arguments passed to pkgreg_remove.

Description

setGEDMethod register a deconvolution algorithm in the CellMix registry.

removeGEDMethod removes a deconvlution algorithm from the registry. Note that it does not delete any NMF algorithm that might be related to it.
