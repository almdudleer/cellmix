Description

Internal Functions for the CellMix Dataset registry

setGEDData register a dataset in the CellMix registry.

removeGEDData removes a dataset from the registry. Note that it does not delete any local/cache file related to this dataset.

meanBasis builds a basis matrix for a data entry from one of its phenotypic data. It tries to match the order of a coefficient matrix if any, or use the order of the variable's levels, which must have corresponding phenotypic variables.

pdataCoef builds a coefficient matrix for a data entry from one of its phenotypic variables, whose levels must correspond to numeric phenotypic variables, that each contains proportions of one cell type that is present in each samples.

pureCoef builds a coefficient matrix for a data entry where all samples are pure, using one of its phenotypic -- factor -- variable whose levels define the cell type of origin of each sample.

gedDataCheck check if all or a given registered dataset can be loaded.

.makeData remake some rda files shipped with CellMix.
