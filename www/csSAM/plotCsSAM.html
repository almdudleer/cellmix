plotcsSAM

Usage

plotCsSAM(csSAMdata, SAMdata, alternative, cellID, numcell, fileName)

Arguments

csSAMdata
List object output of the fdrCsSAM function
SAMdata
List object output of the fdrSAM function
alternative
Type of test conducted. Will appear in plot title.
cellID
Label for each cell-type
numcell
Number of different cell-types being considered.
fileName
Name of output pdf file.

Description

Plots the # of genes called significnat at a given false disocvery rate for the SAM (heterogenous tissue) comparison, and for each of the contrasted cell-types using csSAM

References

Shen-Orr SS, Tibshirani R, Khatri P, Bodian DL, Staedtler F, Perry NM, Hastie T, Sarwal MM, Davis MM and Butte AJ (2010). "Cell type-specific gene expression differences in complex tissues." _Nature methods_, *7*(4), pp. 287-9. ISSN 1548-7105, , .

Author
